# Repo of lhan.me Blog

## How to run

1. Install Node.js

2. Install Hexo

```bash
npm install -g hexo-cli
```

3. Instal theme

What I use is [Typography](https://github.com/SumiMakito/hexo-theme-typography)

4. Generate static contents

```bash
hexo generate
```

5. Deploy the blog

```bash
hexo deploy
```
